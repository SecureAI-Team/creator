# VNC 远程浏览器使用指南

## 用途
当创作者需要手动登录平台但不熟悉 VNC 操作时，提供分步骤的图文指导。
可通过 `/vnc` 触发，或在 `/login` 流程中用户表示不会操作时自动调用。

## 指南内容

### 发送以下分步指引给创作者：

```
📱 VNC 远程浏览器使用指南

VNC 让你可以通过手机或电脑的浏览器，远程操作服务器上的浏览器来完成登录。

━━━ 步骤 1：打开 VNC ━━━

在你的手机或电脑浏览器中打开以下网址：
http://<ECS公网IP>:6080/vnc.html

（如果打不开，请确认 ECS 安全组已开放 6080 端口）

━━━ 步骤 2：输入密码 ━━━

页面会要求输入 VNC 密码。
输入你在部署时设置的密码，点击 Connect。

━━━ 步骤 3：操作浏览器 ━━━

连接成功后，你会看到一个远程桌面，上面已经打开了浏览器。

- 鼠标操作和你本地一样：点击、输入、滚动
- 如果是手机操作：单击=点击，长按=右键
- 可能需要等几秒让页面加载

━━━ 步骤 4：完成登录 ━━━

浏览器已自动打开了你要登录的网站。
按正常方式登录即可：
- 扫码（微信/公众号/视频号）
- 账号密码
- 手机验证码

━━━ 步骤 5：告诉我 ━━━

登录完成后，回到我们的对话，告诉我「登录完成」。
我会自动验证登录状态。

━━━ 常见问题 ━━━

❓ 打不开 VNC 网页
   → 检查 ECS 安全组是否开放了 6080 端口

❓ 输入文字困难
   → VNC 页面左侧有虚拟键盘按钮，可以点击使用

❓ 画面模糊
   → 正常现象，不影响登录操作

❓ 连接断开
   → 刷新页面重新连接即可，登录状态不会丢失
```

### 根据平台提供额外提示

**微信系（公众号/视频号）**：
```
💡 微信提示：公众号和视频号需要用微信扫码登录。
请提前打开手机微信，准备好扫码。
```

**Google 系（YouTube/NotebookLM/Gemini）**：
```
💡 Google 提示：可能会要求两步验证。
请确保手机能收到 Google 验证通知。
```

**国内平台（B站/抖音/小红书）**：
```
💡 提示：可以用手机号+验证码登录，通常比密码方式更方便。
```

# 撤回/删除已发布内容

## 用途
当创作者需要撤回或删除已发布到某平台的内容时，检查平台能力并引导操作。

## 各平台能力

| 平台 | 删除已发布 | 编辑已发布 | 备注 |
|------|-----------|-----------|------|
| 公众号 | ✅ 可删除 | ❌ 不可改 | 群发后不可修改，可删除 |
| 小红书 | ✅ 可删除 | ✅ 可编辑 | 编辑后需重新审核 |
| B站 | ✅ 可删除 | ✅ 可编辑 | 视频可替换、信息可修改 |
| 抖音 | ✅ 可删除 | ❌ 有限 | 视频不可替换，描述可改 |
| YouTube | ✅ 可删除 | ✅ 可编辑 | 标题/描述/标签均可改 |
| 视频号 | ✅ 可删除 | ❌ 有限 | |

## 操作步骤

### 1. 确认目标
解析创作者意图，确定：
- 目标平台
- 要删除/编辑的内容（标题或关键词匹配发布记录）
- 操作类型：删除还是编辑

### 2. 查找发布记录
读取 `workspace/content/published/`，找到匹配的发布记录。
展示匹配结果让创作者确认：

```
找到以下发布记录：
1. 小红书 -「AI 工具推荐」- 2026-02-09 发布
2. 公众号 -「AI 工具推荐」- 2026-02-09 发布

你要操作哪个？输入序号或平台名。
```

### 3. 检查平台能力
根据上表检查目标平台是否支持该操作。
如果不支持：
```
❌ <平台> 不支持修改已发布的内容。
你可以选择：删除后重新发布。
```

### 4. 二次确认（删除操作）
删除是不可逆操作，必须严格确认：

```
⚠️ 确认删除

你确定要从 <平台> 删除以下内容吗？
标题：<标题>
发布时间：<时间>

此操作不可撤销。请回复「确认删除」继续。
```

### 5. 执行 RPA
通过 Browser RPA 操作对应平台后台：
1. 打开平台 profile 浏览器
2. 导航到内容管理页面
3. 找到目标内容
4. 执行删除/编辑操作
5. 确认操作完成

### 6. 更新记录
在发布记录文件中标注已删除：
```markdown
- 状态: 已删除
- 删除时间: <时间戳>
```

### 7. 反馈
```
✅ 已从 <平台> 删除「<标题>」。
发布记录已更新。
```

## 异常处理
- 未登录 → 提示 /login
- 内容未找到（可能已手动删除）→ 通知创作者
- 删除失败 → 截图 + 建议手动操作

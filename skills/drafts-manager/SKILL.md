# 草稿管理

## 用途
当创作者执行 `/drafts` 时，展示所有已保存的草稿列表。

## 操作步骤

### 1. 读取草稿目录
扫描 `workspace/content/drafts/` 下的所有 .md 文件。

### 2. 解析草稿元数据
从每个草稿文件中提取：
- 工具来源
- 创作时间
- 内容类型
- 标题/摘要

### 3. 展示列表
按时间倒序展示：

```
📝 草稿列表

1. [文本] AI Agent 如何改变日常工作流 — ChatGPT
   2026-02-10 14:00 | 未发布
   
2. [视频] 量子计算入门 — NotebookLM
   2026-02-09 10:30 | 已发布到 B站
   
3. [文本] 10 个提升效率的 AI 工具 — Claude
   2026-02-08 16:00 | 已发布到 公众号, 小红书

输入序号查看详情，或告诉我下一步操作。
```

### 4. 草稿操作
- 查看详情：展示完整内容
- 继续编辑：重新打开对应工具的对话
- 发布：进入发布流程
- 删除：移除草稿文件

# 微博发布 RPA Skill

## 概述
通过浏览器自动化操作微博 (https://weibo.com)，完成图文或视频发布。
微博支持：普通微博（文字+图片/视频）、头条文章（长文）。

## 前置条件
- Browser Profile `weibo` 已创建且已登录
- 已有适配后的内容

## 输入
- `content`: 微博正文（≤2000 字）
- `content_type`: "post"（普通微博）或 "article"（头条文章）
- `images`: 图片路径列表（≤9 张，普通微博）
- `video_path`: 视频文件路径（可选）
- `tags`: 话题标签列表
- `article_title`: 头条文章标题（文章类型时）
- `article_content`: 头条文章正文（文章类型时）

## RPA 操作步骤

### 1. Preflight 检查
profile=weibo，导航到微博，检查登录态。
未登录 → 中断，提示 /login weibo。

### 2a. 发布普通微博
- 导航到 `https://weibo.com` 首页
- 找到微博发布框（顶部编辑区域）
- 填入正文内容
- 如果有话题，在正文中加入 #话题# 格式
- 如果有图片，点击「图片」按钮，逐张上传
- 如果有视频，点击「视频」按钮，上传视频文件
- **进度反馈**：视频上传时每 30 秒通知进度

### 2b. 发布头条文章
- 导航到 `https://weibo.com/ttarticle/p/show`  或通过微博头条文章编辑入口
- 填入文章标题
- 在富文本编辑器中填入正文
- 设置封面图
- 添加话题标签

### 3. 设置可见性
- 默认选择「公开」
- 可选「仅自己可见」用于预览

### 4. 添加超话/话题
- 在内容中添加 #超话# 或 #话题#
- 微博话题对曝光有重要影响

### 5. 预览确认
截图 → 发送给创作者确认。

### 6. 发布
创作者确认后点击「发送」/「发布」按钮，等待完成。

### 7. 验证 + 记录
检查发送成功提示，获取微博 URL，截图结果，写入发布日志。

## 微博特殊说明
- 微博是舆论场，发布时间很关键（工作日早 8-9 点，午 12-1 点，晚 9-10 点）
- 带话题 # 和 @大V 可增加曝光
- 图片建议使用高质量原图，微博会压缩
- 视频发布后可能需要等待审核

## 异常处理
- 图片上传失败 → 重试
- 内容包含敏感词 → 截图通知
- 频率限制 → 通知创作者稍后发布

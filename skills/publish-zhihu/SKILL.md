# 知乎发布 RPA Skill

## 概述
通过浏览器自动化操作知乎 (https://www.zhihu.com)，完成文章或视频发布。
知乎支持两种主要发布形式：文章（专栏）和视频。

## 前置条件
- Browser Profile `zhihu` 已创建且已登录
- 已有适配后的内容

## 输入
- `title`: 标题（≤100 字）
- `content`: 正文内容（文章类，Markdown/富文本）
- `tags`: 话题标签列表（≤5 个）
- `content_type`: "text"（文章）或 "video"
- `video_path`: 视频文件路径（视频类）
- `cover_path`: 封面图路径（可选）
- `column`: 专栏名称（可选，发布到特定专栏）

## RPA 操作步骤

### 1. Preflight 检查
profile=zhihu，导航到知乎，检查登录态。
未登录 → 中断，提示 /login zhihu。

### 2. 进入发布页面
- 文章：导航到 `https://zhuanlan.zhihu.com/write`
- 视频：导航到 `https://www.zhihu.com/creator/video/upload`

### 3a. 发布文章
- 填入标题
- 在富文本编辑器中填入正文内容
- 知乎编辑器支持 Markdown，可通过粘贴自动转换
- 设置封面图（如果提供）

### 3b. 上传视频
- 使用 `browser upload` 上传视频文件
- 等待上传和处理完成
- **进度反馈**：每 30 秒通知上传进度
- 填入标题和描述

### 4. 添加话题
- 在话题输入区搜索并添加相关话题
- 知乎话题对 SEO 和推荐有重要影响
- 逐个搜索话题关键词并从下拉列表中选择

### 5. 选择专栏（可选）
- 如指定发布到某专栏，选择对应专栏
- 否则发布为独立文章

### 6. 预览确认
截图 → 发送给创作者确认。

### 7. 发布
创作者确认后点击「发布」按钮，等待完成。

### 8. 验证 + 记录
检查成功提示，获取文章/视频 URL，截图结果，写入发布日志。

## 知乎特殊说明
- 知乎文章有很强的长尾搜索流量，建议标题包含关键词
- 文章内容建议≥1500字，深度内容获得更高权重
- 话题选择关联到推荐池，选热门但精准的话题
- 建议在回答中关联文章以获取额外流量

## 异常处理
- 内容审核 → 截图通知，知乎审核较严格
- 频率限制 → 通知创作者稍后重试
- 编辑器加载异常 → 刷新重试

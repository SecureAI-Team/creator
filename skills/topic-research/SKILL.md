# 选题与热点研究

## 用途
当创作者执行 `/topic [领域]` 或自然语言描述选题需求时，分析热点趋势并推荐选题。

## 操作步骤

### 1. 理解需求
解析创作者输入，确定：
- 领域/方向（如 "AI"、"教育"、"科技"）
- 内容类型偏好（图文/视频）
- 目标平台（如有指定）

### 2. 收集信息来源

#### 方式 A：基于记忆分析（优先）
读取 Workspace 中的：
- `workspace/data/` — 各平台历史数据，识别高表现话题
- `workspace/content/published/` — 已发布内容记录，避免重复
- `workspace/topics/` — 历史选题库

#### 方式 B：实时热点抓取（可选）
如果需要最新热点，使用 browser 工具抓取趋势页面：
- 微博热搜：https://s.weibo.com/top/summary
- B站热门：https://www.bilibili.com/v/popular/all
- 抖音热点：https://www.douyin.com/hot
- 知乎热榜：https://www.zhihu.com/hot

对每个平台：
1. browser navigate 到热点页面（使用通用 profile 或对应平台 profile）
2. browser snapshot 提取热点列表
3. 解析热门话题/关键词

### 3. 生成选题建议
用 Qwen 综合以上信息，生成 5-8 个选题建议。每个选题包含：
- **标题建议**：2-3 个标题变体
- **角度切入**：从什么独特视角展开
- **内容形式**：适合图文还是视频
- **目标平台**：最适合发布到哪些平台
- **热度评估**：当前话题热度（高/中/低）
- **竞争评估**：同类内容多不多

### 4. 展示与交互
将选题建议列表展示给创作者：

```
📋 选题推荐（领域：AI）

1. 🔥 AI Agent 如何改变日常工作流
   角度：从个人实践出发，展示 3 个真实场景
   形式：图文 | 适合：公众号、知乎、小红书
   热度：高 | 竞争：中

2. 💡 用 AI 做副业的 5 种可行方式
   角度：实操指南，每种附带月收入预期
   形式：图文/视频 | 适合：全平台
   热度：高 | 竞争：高

...

选一个？或者告诉我你想要的方向，我再细化。
```

### 5. 记录选题
将创作者选择的选题保存到 `workspace/topics/` 下：

```markdown
# 选题记录

- 日期: 2026-02-10
- 领域: AI
- 选题: AI Agent 如何改变日常工作流
- 角度: 个人实践 + 3 个真实场景
- 状态: 待创作
- 来源: 热点分析
```

# ChatGPT 提示词模板：长文创作

## 用途
用于通过 ChatGPT Deep Thinking 生成深度文章。

## 模板

```
请你作为一位资深的自媒体内容创作者，帮我写一篇关于「{topic}」的深度文章。

要求：
- 字数：{word_count} 字左右
- 风格：{style}
- 受众：{audience}
- 结构：引言 + 3-5 个核心观点 + 总结
- 每个观点配 1-2 个具体案例或数据支持
- 标题要有吸引力，适合社交媒体传播

{additional_requirements}
```

## 变量说明
- `{topic}`: 文章主题/选题
- `{word_count}`: 目标字数（默认 2000）
- `{style}`: 写作风格（默认 "专业但通俗易懂"）
- `{audience}`: 目标受众（默认 "对该领域感兴趣的普通读者"）
- `{additional_requirements}`: 创作者的额外要求
